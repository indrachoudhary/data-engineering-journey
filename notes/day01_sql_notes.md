# Day 01 - SQL Basics

## What is SQL?
SQL (Structured Query Language) is a declarative language used to interact with relational databases.

## SELECT
Used to retrieve specific columns from a table.

## WHERE
Filters rows before grouping or aggregation.

## GROUP BY
Groups rows based on one or more columns to apply aggregation functions.

## HAVING
Filters aggregated results after GROUP BY.

## Aggregation Functions
- SUM()
- COUNT()
- MAX()
- MIN()
- AVG()

## Key Learning
Understanding aggregation and filtering is foundational for data engineering.
